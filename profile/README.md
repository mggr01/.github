- 사이드 프로젝트를 통한 실무적 능력 함양

## 🌟계획
### 1. 주제
- Azure 기초 공부 및 AZURE 104 자격증 준비
  - 책 : 처음배우는애저
  - 하루 1단원씩 발표 형식 실습 진행
  - 마지막 주는 자격증 대비 덤프 풀기
  - 개인 공부 필수 (덤프 풀기 및 질문)
- AZURE 기반 사이드 프로젝트(자격증 취득후 )
- 토론 및 질문
